Special Numbers in Js:
=> Infinity, -Infinity, -0, NaN 
=> NaN : Not a Number. It is used for Mathematical Operations
=> Infinity and Nan are "Number" datatypes.

Floating Point Numbers in Js:
=> Js treats decimals as floating point numbers.
=> addition, multiplication are subject to rounding errors.

Math Functions:

1. Math.trunc() - returns only integer part.
2. Math.round() - returns round-off values.
3. Math.floor() - returns the lower round-off value.
4. Math.ceil() - return the upper round-off value.

NOTE: trunc() and floor() works differently with -ve numbers.

5. Math.max() - Returns the maximum value.
6. Math.min() - Return the minimum value.
7. Math.pow(base, power) - Returns the power of a number.

8. Math.PI = 3.141592653589793
