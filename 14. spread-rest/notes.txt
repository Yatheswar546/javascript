Spread Syntax (...)

=> In Pass by Reference we can't create a copy for arrays and objects but we need to original data.
=> In such cases, we use Spread Operator (...)

Eg: let x = [1,2,3]  // #zzz  --> [1,2,3]
    let y = [...x]   // #yyy  --> [1,2,3]

=> In the above example a copy is created rathen than Reference.
=> Any changes made in New Data will not effect original data.
