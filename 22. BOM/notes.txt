Browser Object Model (BOM):

=> It represents the current browser window. The main purpose of BOM is to manipulate 
    the properties and methods which are associated with the browser window.

                                window
        ___________________________|___________________________
        |            |             |             |             |
     document     history       location       screen       navigator

=> document object  : Represents html structure and content in tree format.

=> history object   : data of URLs.

=> location object  : info. about current URL.

=> screen object    : info. about browser screen like width/height.

=> navigator object : info. about user's browser & OS.
