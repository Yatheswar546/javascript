Questions:

1. difference between null and undefined?
Answer:
When a variables is declared but not assigned any value then it is "undefined".
When a variable is assigned with Null value then it is "null".

Eg: 
var a;  // undefined
var b = null; // null

2. why typeof(null) is an object?
Answer:
The reason typeof null returns 'object' in JavaScript is a historical bug 
from the language's original implementation that has been preserved for backward compatibility.
