// Keydown event

document.body.addEventListener('keydown', function(e) {
    var keycode = e.keyCode;

    // 13 is keycode for enter
    if(keycode === 13){
        console.log(keycode + 'KeyDown - A key is clicked !!!')
    }
    // console.log(keycode + 'KeyDown - A key is clicked !!!')

Js - KeyBoard Events:

Events triggered when user clicks a key.

3 types:

1) keydown   : key is pressed.

2) keyup     : key is released after it's keydown event.

3) keypress  : when any key is in pressed position (except SHIFT, FN, CAPS LOCK).
